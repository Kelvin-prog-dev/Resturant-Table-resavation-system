 
Restaurant Table Reservation System

A PHP-based web application for managing restaurant table reservations. The project includes form processing for reservation submissions, customer information handling, and database integration for storing reservation and customer data. The system features input validation and error handling to ensure data integrity.

### Main Function Points
- Restaurant table reservation form submission and processing
- Customer information collection and storage
- Reservation data management with database integration
- Input validation and error handling for form submissions
- Restaurant information display page

### Technology Stack
- PHP (49.1%) - Backend server-side logic
- Hack (41.3%) - Additional backend functionality
- CSS (9.6%) - Frontend styling
- SQL/Database - Data persistence for customers and reservations
